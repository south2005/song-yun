# 宋韵流芳 - 宋代书法文化互动游戏

## 项目简介

"宋韵流芳"是一款基于HTML5的宋代书法文化互动游戏，旨在通过游戏方式弘扬宋代书法文化。游戏融合了NPC对话互动和书法找茬游戏元素，玩家可以与宋代著名书法家对话交流，学习书法知识，并通过找出书法作品中的差异来完成任务。

## 主要功能

1. **宋代场景与NPC互动**：
   - 玩家可以与苏轼、黄庭坚等宋代著名书法家对话交流
   - 通过对话了解宋代书法的历史、特点和文化背景
   - NPC会布置书法找茬任务，并在完成后提供更多书法知识

2. **书法找茬游戏**：
   - 比对原作与临摹作品，寻找书法中的差异
   - 支持《寒食帖》、《松风阁诗帖》等宋代名作
   - 提供计时挑战和提示功能
   - 游戏结束后获得相关书法知识奖励

3. **书法知识学习**：
   - 收集"宋四家"（苏轼、黄庭坚、米芾、蔡襄）的书法知识
   - 了解宋代书法的艺术特点和文化价值
   - 学习宋代书法与文人精神的关系

## 使用说明

1. 打开index.html文件启动游戏
2. 在开始界面点击"开始游戏"进入
3. 阅读游戏规则后点击"进入"
4. 与NPC对话，选择对话选项
5. 接受找茬任务，在临摹作品中找出与原作的差异
6. 完成任务后继续对话，探索更多宋代书法知识

## 游戏模块
本项目包含一个主要的游戏模块：

**妙笔生花 (书法文字找茬)**
  - 玩家需要在两幅宋代书法作品中找出7处细微差异
  - 游戏随机选择苏轼的《寒食帖》或《松风阁诗帖》作为游戏素材

  - 当玩家找出所有差异时，游戏结束并计算得分，包括找茬得分和时间奖励
  - 游戏设有提示功能，但使用提示会扣除10秒的游戏时间
  - 完成游戏后，玩家可以了解当前书法作品的历史背景和书法特点
  - 玩家可以随时返回主页或选择再玩一次

# 妙笔生花 - 书法找茬游戏

## 项目介绍

《妙笔生花》是一款专注于宋代书法鉴赏的互动游戏，旨在提高用户对书法细节的观察力和鉴赏能力。游戏设计精美，融合了传统文化元素，让玩家在游戏中感受宋代书法艺术的魅力。

## 游戏特色

- **经典书法赏析**：精选《寒食帖》、《松风阁诗帖》等宋代名家书法作品
- **找茬游戏玩法**：通过在两幅相似书法作品中找出差异，提升观察力
- **文化知识普及**：每幅作品都附有详细的历史背景和艺术价值介绍
- **全屏沉浸体验**：精美的传统山水画背景，营造古典氛围
- **计时挑战模式**：限时找出所有差异，增加游戏趣味性和挑战性

## 游戏规则

1. 仔细比对左右两幅书法作品，寻找字形差异
2. 在两幅书法作品中寻找不同之处，发现后点击即可标记
3. 如果遇到困难，可以点击"提示"按钮获取帮助（但会扣除10秒游戏时间）
4. 当找出所有5处差异或时间耗尽时，游戏结束
5. 游戏结束后会显示你的得分和完成情况
6. 可以选择"再玩一次"继续游戏，或"返回主页"退出游戏

## 技术实现

- 前端：HTML5 + CSS3 + JavaScript
- 响应式设计：适配不同屏幕尺寸
- 动态生成游戏内容：随机选择不同的书法作品进行挑战
- 差异检测算法：准确识别用户点击位置是否为差异点

## 如何使用

1. 点击开始游戏进入主界面
2. 选择一幅书法作品开始挑战
3. 在规定时间内找出所有差异
4. 游戏结束后查看成绩并了解作品的历史背景
