# 墨子国家级博物馆网站

这是墨子国家级博物馆的官方网站源代码，展示了墨子的生平、科学成就和思想理论。

## 如何运行

1. 确保你已安装Python 3或其他HTTP服务器
2. 在项目根目录下打开命令行，运行：
   ```
   python -m http.server 8000
   ```
   或者使用任何其他HTTP服务器软件
3. 在浏览器中访问 http://localhost:8000

## 网站结构

- `index.html` - 主网页文件
- `style.css` - 样式表文件
- `script.js` - JavaScript交互脚本
- `images/` - 网站所需的图像资源

## 注意事项

- 如果图片无法正常显示，请确保`images`目录包含所有必要的图片文件
- 网站设计采用了中国传统水墨风格，优化了响应式显示
- 主要内容包括：墨子生平、科学成就、思想理论和现代传承等展览内容
